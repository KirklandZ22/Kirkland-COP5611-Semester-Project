Setting up a virtual machine will allow for more customization then what is presented here.

Below are the steps to setup a virtual machine:

  • Before starting ensure that Virutalization is enabled in BIOS.
    • Mac Virutalization
    • Windows Virutalization
  • Download and install a hypervisor. Some popular hypervisors are:
    • VMware Workstation Player
    • VirtualBox
  • Find the operating system you wish to run in the virutal machine. Some ways to do this are:
    • distrochooser.de
    • distrowatch.com
  • Or to go directly to some of the one shown here
    • KolibriOS
    • Damn Small Linux
    • Debian
    • Ubuntu
  • Follow the steps in the hypervisor to run the virutal machine and enjoy.
